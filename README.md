### What I Build

I build **autonomous AI systems** that think, decide, and execute 24/7 without human intervention.

My current project — **MAX HQ** — is a JARVIS-style AI command center running 6 departments, 39 specialized agents, and 24 automated schedules. It handles research, marketing, security audits, content generation, and system operations while I sleep.

Previously: Senior Data Analyst who taught himself Python and LLMs. Now building the infrastructure layer for AI agents.
